# weather-widget #

A demo application for displaying weather information.

## Environment Variables ##

There are a couple environment variables used within this application:

* MOCK_WEATHER [ true | false ] => This states if this application should use mock weather data (if true), or if
it should use actual weather data currently supplied by weather undergrounds API (if false).
* WUNDERGROUND_API_KEY [ api_key ] => This API key is used to connect to weather underground to retrieve weather
 data.
